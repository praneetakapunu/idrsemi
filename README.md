# IDRSEMI - Open Source Chip Design Project

## Overview
This project aims to create a completely open-source chip design that can be taped out using freely available tools and foundries.

## Goals
- Design a functional semiconductor chip using only open-source tools
- Document the complete design flow from concept to tapeout
- Create reproducible results for the open hardware community
- Target an open PDK (Process Design Kit) for manufacturing
